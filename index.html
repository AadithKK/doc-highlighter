Features

Set it up once.
Highlight everything.

Build your queue, press start, and watch every word light up — no clicking required.

🎨

8 highlight colours

Yellow, Green, Blue, Red, Purple, Orange, Teal, and Pink — each calibrated to a swatch in the Google Docs colour palette. Mix colours freely across your queue.

8
highlight colours
📋

Batch import

Paste a block of text — one entry per line. Assign colours inline with the word::Color syntax. Import dozens of terms in one shot.

🎯

5-second calibration

Click Capture, hover over each colour swatch in Google Docs, and the coordinates are saved. Do it once — works every session at that zoom level.

⏱️

Countdown timer

3, 5, 10, or 15 seconds to switch back to your doc before automation begins.

🛡️

Failsafe

Move the mouse to the top-left corner at any time to instantly abort. No keyboard needed.

📊

Progress bar

Live count of entries processed with a smooth progress bar and status label.

∞
Free forever
⚡

Adjustable delay

Slide from 0.2s to 2.0s between actions. Bump it up on slower connections so Google Docs has time to register each highlight before the next word is searched.

Installation

Running in under a minute.

Three commands. No config files, no accounts.

1

Clone the repo

git clone https://github.com/AadithKK/doc-highlighter.git cd doc-highlighter
2

Install dependencies

pip install customtkinter pyautogui pillow
3

Run it

python doc_highlighter.py
4

Calibrate (one-time setup)

Go to Settings → Calibration. Open your Google Doc, click the highlight button, and use CAPTURE to record each colour swatch position. Takes about 60 seconds total — never needed again.

AI Integration

Claude builds the list.
You press Start.

Ask Claude Code to extract key terms from any text, then send them straight to Doc Highlighter with a single curl command. The queue fills and highlighting begins — no copy-paste, no switching windows.

Get Started
Add one term
curl -X POST http://localhost:7798/add \ -d "mitochondria::Yellow"
Batch import + start immediately
curl -X POST http://localhost:7798/batch \ -d "ATP synthesis::Yellow cell membrane::Green DNA replication::Blue"
Add and start at once
curl -X POST http://localhost:7798/add-and-start \ -d "key concept::Orange"
