Get Started
Up and Running
in Minutes
01

Clone the Repo

Grab the source from GitHub and enter the project directory.

git clone https://github.com/AadithKK/doc-highlighter.git
cd doc-highlighter
02

Install Dependencies

All Python packages needed for the GUI, automation, and HTTP server.

pip install customtkinter pyautogui Pillow
03

Launch the App

Run the script — the GUI opens and the HTTP API server starts automatically on port 7798.

python doc_highlighter.py
04

Add Your First Highlight

Use the GUI or send your first curl command. Focus your Google Doc, then press Start.

# One line = one highlight (uses default color)
curl -s -X POST http://localhost:7798/add-and-start \
  -H "Content-Type: application/json" \
  -d '{"text":"hello world","color":"Yellow"}'
Claude Code Integration
Let AI Drive
Your Highlights
HTTP API

Every Action,
One curl Away

Doc Highlighter ships with a built-in HTTP server. Claude Code (or any script) can add terms, start the queue, check status, and clear the list — no GUI interaction required.

POST /add
Add a single highlight entry to the queue
POST /add-and-start
Add an entry and immediately start the queue
POST /batch
Add multiple highlights at once (newline-separated)
POST /start
Start processing the current queue
POST /stop
Stop the queue (can be resumed)
GET /status
Returns running state, queue count, current entry
Batch Example
# Highlight multiple terms with colors in one request curl -s -X POST http://localhost:7798/batch \   -H "Content-Type: application/json" \   -d '{"lines":"transformer architecture::Yellow\nattention mechanism::Blue\ngradient descent::Green","color":"Yellow"}'
